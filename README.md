🤖 PyCursor: Self-Healing Hybrid AI Agent
A sophisticated autonomous coding agent that translates natural language into executed Python code.

Core Tech: LangGraph, Ollama (Qwen2.5-Coder), Google Gemini 1.5, Python.

The Problem: LLMs often produce code with syntax or indentation errors that require manual fixing.

The Solution: Built a state-machine driven agent that catches terminal errors (Tracebacks), feeds them back into the LLM, and self-corrects the code in real-time.

Key Innovation: Implemented a Hybrid Switch allowing 100% private local execution (Ollama) or high-performance cloud processing (Gemini).
